# 🌍 TerraScore AI

Climate + Drought Prediction System for Farmers

## Features
- NDVI (Vegetation Health)
- LST (Temperature)
- Drought Prediction (ML Model)
- Farmer Credit Score

## Tech Stack
- FastAPI (Backend)
- Streamlit (Frontend)
- XGBoost (ML)
- Google Earth Engine
