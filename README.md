# TerraScore
AI-Powered Agricultural Credit Scoring Platform

TerraScore is a geospatial scoring platform that combines Earth observation data with project-defined scoring logic.

## Current Status

This repository now includes:

- Framework-based structure (`frontend` with React + Vite)
- Required documentation (`ERD`, flow diagram, API docs, schemas)
- Dedicated Google Colab workspace file
- Dataset verification blocker documentation and next steps

## Repository Structure

- `frontend/` React application (Vite)
- `backend/` backend API placeholder and notes
- `docs/` architecture and process documentation
- `schemas/` JSON schemas for core entities
- `data/` data pipeline notes and CSV output location
- `colab/` Google Colab notebooks

## Quick Start

### Frontend

1. `cd frontend`
2. `npm install`
3. `npm run dev`
